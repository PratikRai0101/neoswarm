## NeoSwarm

NeoSwarm is a fork of OpenSwarm that extends the original with:

- **Linux-first support** — Replaced Electron shell with Tauri v2 (Rust)
- **Native AgentLoop** — Replaced proprietary `claude-agent-sdk` with native Python implementation
- **Ollama support** — Added local model inference via Ollama
- **Orchestrator Agent** — Team Lead agent for task decomposition and worker management
- **CLI + TUI** — Added terminal interfaces alongside GUI
